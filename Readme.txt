Axis Saral portal

Steps to start backend:-
1. First import backend as maven project.
2. Create A Database with the name as given in application.props file in backend.
3. To initalize the Email service Provide the email and go to google account and create new app password.
4. And provide that email and that new app password in application.props file for email service.
5. Then run the backend application.

Steps to start frontend:- 
1. Open frontend in any code editor.
2. Then run command in terminal "npm install" to download node modules.
3. Then run command in terminal "npm start" and frontend will start.
